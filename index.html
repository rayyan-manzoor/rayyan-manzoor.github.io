JANA - Health Tracker

Simplifying daily health and medication management.

JANA is a concept mobile application focused on improving patient adherence to complex medication schedules. The design uses a minimalist, dark UI consistent with the portfolio theme for low-light use.

Key Features:

  • Intuitive pill tracking and reminder system.
  • Integration with wearable devices for vital sign logging.
  • Secure cloud-based data storage for doctor consultation.

This project involved UX/UI design, wireframing, and creating high-fidelity mockups in Figma.
